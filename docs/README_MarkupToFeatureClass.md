# Markup to Feature Class

This tool converts a ArcGIS Field Maps .markup file to GDB feature classes. It any attribute fields available in the .markup file (color, note, etc) are preserved in the output feature classes.

- A CSV that summarizes the files within the user specified directory
- A CSV that summarizes the immediate sub-directories of the user specified directory

![screenshot_SummarizeFolderFiles_3.png](https://raw.githubusercontent.com/mpanunto/PanunTools/main/docs/screenshot_MarkupToFeatureClass_1.png)

### User Inputs

1. Path to markup file
2. Path to output directory
